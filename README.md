# Calculator

A simple browser-based calculator that performs basic arithmetic operations on two numbers.

## Features
- Addition, subtraction, multiplication, and division
- A Fahrenheit button to convert Celsius to Fahrenheit (0 in the calculator will change to 32 once the button is pressed)
- Backspace and clear functionality
- Decimal number support
- Clean, responsive UI

## Built With
- HTML
- CSS
- JavaScript

## How to Use
Enter the first number, select an operation, enter the second number, and press equals.
